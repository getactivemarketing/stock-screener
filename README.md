# Stock Screener

A stock screening application with sentiment analysis from multiple sources, value/momentum scoring, and a real-time dashboard.

## Features
- Multi-source sentiment aggregation (Reddit, Stocktwits, Finviz)
- Value and momentum scoring with AI classification
- Return tracking for backtesting
- SvelteKit dashboard with TradingView charts

## Services
- **Pipeline**: Runs every 30 min during market hours
- **Return Tracker**: Runs daily to calculate historical returns
- **Dashboard**: https://web-dashboard-bice.vercel.app
